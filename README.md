## About Me

**안녕하세요! 소통으로 성장하는 프론트엔드 개발자, 설윤성입니다.**

주로 웹 서비스 개발을 담당했으며 사용자와 유연하게 대화하는 인터페이스 개발을 지향합니다.<br>
